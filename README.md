# 🎧 DJ Music App


## ✨ Features

| Feature | Description |
|--------|-------------|
| 🎛️ **DJ Mixer** | Dual-deck mixer with jog wheels, EQ, pitch fader, crossfader, hot cues & loop pads |
| 🥁 **Drum Pad** | Tap-to-play drum pads for beat creation |
| ✂️ **Ringtone Cutter** | Trim any audio file into a custom ringtone |
| 🎚️ **Audio Mixer** | Mix multiple audio tracks with volume control |
| 🔗 **Sound Merger** | Merge two or more audio files into one |
| 📚 **My Library** | Browse and manage your music library |
| ⚙️ **Settings** | Customizable recording format, audio output, language, and more |

## 🚀 Getting Started

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install) `>=3.0.0`
- Dart SDK `>=3.0.0`
- Android Studio / VS Code
- Android Emulator or physical device (Android 6.0+ / iOS 12+)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/dj_music_app.git

# 2. Navigate into the project
cd dj_music_app

# 3. Install dependencies
flutter pub get

# 4. Run the app
flutter run
```

### Build APK

```bash
flutter build apk --release
```
